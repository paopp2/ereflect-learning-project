Unggoy Type 🙊

was created as an Angular learning project by Mon and Pao. It showcases a typing test that displays the user's WPM (words per minute), accuracy, and errors. Unggoy Type is a localized version of the famous Monkeytype as the creators aimed to create a simple application so that they could apply what they have learned after watching endless Udemy tutorials by Maximilian Schwarzmüller. 😂

TLDR, they were tasked to create a simple project and coincidentally, wanted a project-based type of learning.
